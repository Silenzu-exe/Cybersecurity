
## Useful networking commands:

### 1. IP command:
```bash
# list all the Ip addresses of the system
ip a

```
	
### 2.  ifconfig:
```bash
# this cmd does the exact thing as 'ip a' but ifconfig is consider old school way of listing system ip addresses
ifconfig 

```


### 3.  iwconfig: 
```bash
# iwconfig is used for wireless.
iwconfig
```
