
## Useful networking commands:

### 1. IP command:
```bash
# list all the Ip addresses of the system
ip a

```
	
### 2.  ifconfig:
```bash
# this cmd does the exact thing as 'ip a' but ifconfig is consider old school way of listing system ip addresses
ifconfig 

```


### 3.  iwconfig: 
```bash
# iwconfig is used for wireless.
iwconfig
```

### 4. arp (address resolution protocol) / ip n:

```bash
#this cmd is assosicate with mac address
which tell us which ip is associate with with mac address
arp -a #old school

ip n #this is new school
```

## 5. route/ ip r:

```bash
#This command provides the routing table.
route #cmd

ip r #cmd
```
